This course unit will be assessed based on two practical assignments, the first to be carried out in a group, the second individually.

Assignment 1 - Group (2-3 members)

This work aims to explore deep learning models applied to the GTSRB dataset (German traffic sign dataset) to obtain the best possible accuracy result on the test dataset (the best published result is 99.82%).

The work is divided into two parts:

In the first part, models are supposed to be trained using data augmentation, both in pre-processing and dynamically. The aim of this first part is to explore filters and image processing methods and evaluate their impact on the final network performance. Part of the evaluation will be dictated by the image processing techniques used. Several models should be trained with different data augmentation methods, and an analysis of the results obtained should be performed.
In a second phase, the intention is to study the potential of using network ensembles. The networks that will be part of the ensemble are those trained in the first phase. 
German dataset:

Training images ( ZIP )
Test images ( ZIP )
In addition to the notebook (or standalone Python scripts), we want a short report describing the choices made and presenting the tests performed. The notebook or scripts should also be properly commented.

Assignment 2 - Individual

This assignment aims to have students explore GANs and/or autoencoders in a student-choice problem. Many datasets are available for training this type of network.

The main objective is to explore error functions, for example along the lines of what was done in Pix2Pix or Cycle GAN, and/or to explore manipulations and combinations of datasets.

The report should be concise, possibly included in the notebook itself, and should contain a description of the strategy defined by the students as well as a discussion/comparison of the results obtained. In the case of comparison, this should be based on results published by third parties.

Submission via BB.
